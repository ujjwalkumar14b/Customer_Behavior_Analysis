# Customer Behavior Analysis
An end-to-end data analysis project exploring customer shopping patterns, demographic spending habits, discount utilization, and loyalty behaviors using SQL, Python (Jupyter Notebook), and Power BI.

## Project Overview
Understanding customer behavior is essential for optimizing marketing strategies, product placements, and retention programs. This project analyzes transactional customer data to answer key business questions around customer segmentation, revenue drivers, product popularity, and subscription trends.

## Repository Structure
```
├── Customer_Behavior_Analysis.ipynb   # Exploratory Data Analysis & Python Work
├── Customer_Behavior_Analysis.sql     # SQL Queries for business insights
├── Customer_Behavior_Dashboard.pbix   # Interactive Power BI Dashboard
├── Customer_Shopping_Behavior.csv     # Raw dataset
├── customer_sql.txt                   # SQL script backup / quick reference
└── README.md                          # Project Documentation
```

## Key Business Questions & SQL Analytics
The repository includes SQL scripts (Customer_Behavior_Analysis.sql) that address several critical business metrics:

- Gender Revenue Breakdown: Calculates total revenue generated by male vs. female customers.
- High-Value Discount Users: Identifies customers who used discounts but still spent above the average purchase amount.
- Top Rated Products: Retrieves the top 5 products with the highest average review ratings.
- Shipping Method Comparison: Compares average purchase amounts between Standard and Express shipping options.
- Subscription Impact: Evaluates whether subscribed customers generate higher revenue and spend more on average compared to non-subscribers.
- Discount Reliance by Product: Identifies the top 5 products with the highest percentage of purchases using discounts.
- Customer Segmentation: Classifies customers into New, Returning, and Loyal segments based on purchase history.
- Category Top Performers: Uses window functions (ROW_NUMBER()) to identify the top 3 most purchased products per category.
- Repeat Buyer Subscription Likelihood: Analyzes whether frequent buyers (>5 past purchases) tend to subscribe.
- Age Group Revenue Contribution: Aggregates total revenue contributed across different age brackets.

## Tools & Technologies Used
- SQL (PostgreSQL): Aggregations, Window Functions, Common Table Expressions (CTEs), Conditional Logic (CASE statements).
- Python (Jupyter Notebook): Data cleaning, feature engineering, and exploratory data analysis (EDA).
- Power BI: Interactive data visualization and dashboard design.
- Git/GitHub: Version control and project presentation.

## Getting Started
- Clone the Repository: git clone [https://github.com/ujjwalkumar14b/Customer_Behavior_Analysis.git](https://github.com/ujjwalkumar14b/Customer_Behavior_Analysis.git)
- Database Setup: Import Customer_Shopping_Behavior.csv into your PostgreSQL/SQL database into a table named customer. Execute the queries in Customer_Behavior_Analysis.sql to reproduce the analytical findings.
- Dashboard: Open Customer_Behavior_Dashboard.pbix in Power BI Desktop to explore interactive visuals.
